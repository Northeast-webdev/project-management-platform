# Project Brief

## Overview
A full-stack project management platform with focus on task management through Kanban boards, with additional features for chat (to be defined) and mind mapping capabilities.

## Core Purpose
Enable users to manage projects, organize tasks in a Kanban board view, and interact through a chat interface. The platform provides a modern, intuitive interface for project and task management.

## Key Features
1. **Project Management**: Create and manage multiple projects
2. **Task Management (Kanban)**: Organize tasks in a drag-and-drop Kanban board with columns (To Do, In Progress, Review, Done)
3. **Task Creation/Editing**: Modal-based interface for creating and editing tasks
4. **Chat View**: Placeholder for future chat functionality (to be defined)
5. **Mind Map**: Basic mind mapping functionality (not current focus)

## Current Focus
- **Primary**: Task Management (Kanban view) and Task Creation Model
- **Secondary**: Integration between frontend and backend for task operations

## Project Structure
- **Frontend**: React application with styled-components
- **Backend**: Node.js/Express REST API
- **Database**: SQLite (default) or MySQL (optional) with support for both

## Technology Stack
- Frontend: React 18, styled-components, react-beautiful-dnd
- Backend: Node.js, Express
- Database: SQLite3 (default) / MySQL (optional)
- State Management: React Context API with useReducer
